# Goviya E-Commerce PHP Web App

Goviya is a simple full-stack e-commerce web application developed for selling fresh vegetables and fruits online. This project was built using PHP, MySQL, HTML, CSS, and JavaScript as a university project.

## Features

### Customer Side

* User registration and login system
* Browse products by categories
* Search and filter products
* Product details page
* Shopping cart management
* Secure checkout process
* Multiple payment options (Card, Cash on Delivery, Bank Transfer)
* Order history tracking
* User profile management
* Password reset system

### Admin Side

* Dashboard with sales statistics
* Product management (Add, Edit, Delete)
* Category management
* Order management and status update
* User management

## Technologies Used

* PHP (Core PHP)
* MySQL
* HTML5
* CSS3
* JavaScript (AJAX)
* PDO Database Connection

## Project Structure

* **index.php** → Homepage
* **includes/** → Configuration and reusable components
* **pages/** → Customer pages
* **admin/** → Admin dashboard and management
* **goviya_db.sql** → Database file

## Installation

1. Clone or download the project.
2. Move project folder into XAMPP htdocs folder.
3. Import `goviya_db.sql` into MySQL using phpMyAdmin.
4. Configure database in `includes/config.php`.
5. Start Apache and MySQL in XAMPP.
6. Open project in browser:

http://localhost/goviya/
